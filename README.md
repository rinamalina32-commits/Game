🎮 Змейка — аркадная игра, разработанная с использованием SFML

📋 Описание проекта
Игра представляет собой аркадный платформер с элементами головоломки. Проект создан в рамках учебной практики и демонстрирует:

Объектно-ориентированное программирование

Работу с графикой и звуком

Систему управления несколькими игроками

Механику уровней и прогресса

🎮 Геймплей
Основные особенности:

Режимы игры: одиночный и многопользовательский

Уровни сложности: три уровня с различной сложностью

Механики: сбор предметов, преодоление препятствий

Графика: 2D спрайты и анимации

🎯 Управление
Одиночный режим: стрелки для перемещения

Многопользовательский режим:

Игрок 1: стрелки

Игрок 2: WASD

Действие: клавиша пробел

🛠 Системные требования
ОС: Windows 7/8/10 или аналоги

Процессор: с поддержкой SSE2

Оперативная память: минимум 2 ГБ

Место на диске: 200 МБ

Видеокарта: с поддержкой DirectX 9

🔧 Установка
bash
# Клонирование репозитория
git clone https://github.com/username/GameProject.git

# Переход в директорию
cd GameProject

# Сборка проекта
mkdir build
cd build
cmake ..
make
